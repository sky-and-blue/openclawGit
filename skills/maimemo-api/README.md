# 墨墨背单词 OpenClaw 技能

📚 基于墨墨开放API封装的学习数据查询工具，快速获取你的背单词进度、待复习单词等信息。

---

## ✨ 功能特性
1. **学习进度查询**：快速查看今日完成率、学习时长
2. **今日单词管理**：查看今日学习单词列表，支持筛选已完成/新学单词
3. **学习记录查询**：统计总单词量、查询指定日期待复习单词
4. **自动格式化输出**：人性化的结果展示，无需解析JSON

---

## 🚀 安装配置
### 自动安装（已完成）
- Token已自动写入`~/.zshrc`环境变量 `MAIMEMO_ACCESS_TOKEN`
- 命令路径已加入系统PATH，新开终端即可直接使用
- 所有命令已添加可执行权限

### 手动配置（如需更新Token）
```bash
# 1. 更新环境变量
echo 'export MAIMEMO_ACCESS_TOKEN="你的新Token"' >> ~/.zshrc
# 2. 生效配置
source ~/.zshrc
```

## 📖 命令使用说明

### 1. `maimemo-progress` - 获取今日学习进度
**功能**：查看今日已完成单词数、目标总数、完成进度、学习时长
**示例**：
```bash
maimemo-progress
```
**输出**：
```
✅ 今日学习进度：
已完成单词：126 个
今日目标：200 个
完成进度：63%
学习时长：42 分 36 秒
```

---

### 2. `maimemo-today` - 获取今日学习单词列表
**功能**：查看今日所有学习单词，支持筛选
**参数**：
- `[limit]` 可选，返回条数上限，默认200
- `--finished true/false` 可选，仅返回已完成/未完成单词
- `--new true/false` 可选，仅返回新学/复习单词

**示例**：
```bash
# 查看所有今日单词（默认200条）
maimemo-today

# 查看今日已完成的10个单词
maimemo-today 10 --finished true

# 查看今日新学单词
maimemo-today --new true
```
**输出**：
```
📋 今日学习单词（共200个）：
- insurance ✅ FAMILIAR
- vehicle ✅ FAMILIAR
- tackle ⏳ 未学习
- brim 🆕 ⏳ FORGET
```
- 🆕 表示新学单词
- ✅ 表示已完成
- ⏳ 表示未完成
- 后面是你对单词的第一次反馈：熟悉/模糊/忘记等

---

### 3. `maimemo-records` - 查询学习记录
**功能**：查询历史学习记录、待复习单词、总单词量
**参数**：
- `--count` 可选，仅返回总学习单词数（速度更快）
- `--date YYYY-MM-DD` 可选，查询指定日期需要复习的单词
- `--limit N` 可选，返回条数上限，默认100

**示例**：
```bash
# 查询总单词量
maimemo-records --count

# 查询明天（2026-03-20）需要复习的单词
maimemo-records --date 2026-03-20

# 查询最近100条学习记录
maimemo-records --limit 100
```
**输出**：
```
📊 总学习单词量：2896 个
```
```
📅 2026-03-20 需要复习的单词（共156个）：
- apple | 学习次数：5 | 下次复习：2026-03-20
- banana | 学习次数：3 | 下次复习：2026-03-20
```

## ❓ 常见问题
### Q: 返回的单词数/进度是null？
A: 请打开墨墨背单词APP，下拉同步一下数据，接口需要APP同步后才能获取到最新数据。

### Q: 接口报错无权限？
A: Token可能过期了，重新获取Token后更新`~/.zshrc`里的`MAIMEMO_ACCESS_TOKEN`即可。

### Q: 最多能返回多少条数据？
A: 接口限制最多返回1000条，超过会自动截断。
